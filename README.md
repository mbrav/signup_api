## FastAPI signup_api

An asynchronous Fast API service for signups and Telegram integration

### Run FastAPI backend in Docker

With docker-compose installed, do:

```bash
docker-compose up
```

Go to [0.0.0.0:8000/docs](http://0.0.0.0:8000/docs) for SwaggerUI
